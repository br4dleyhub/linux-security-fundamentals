# Linux Security Fundamentals

## Overview
This project is a hands-on learning lab focused on understanding fundamental Linux security concepts.

It introduces how users, permissions, processes, and logs contribute to system security in a controlled educational environment.

---

## Learning Objectives
- Understand Linux user and group management
- Learn file permissions and access control
- Observe running processes and services
- Analyze system logs
- Apply basic system hardening concepts
- Reflect on ethical responsibilities

---

## Technologies Used
- Linux (Ubuntu)
- Bash shell
- Core Linux utilities
- Command Line Interface (CLI)

---

## Scope & Ethics
This project is developed strictly for educational purposes.
All actions are performed on a local system owned by the learner.
No real users, production systems, or external machines are involved.

---

## Program Alignment
- Introduction to Cyber Security
- Essentials of IT & Ethics
- Data Communication and Net-Centric Computing

---

## Conclusion
This project provided hands-on experience with Linux security fundamentals,
including access control, process monitoring, logging, and basic hardening.

It highlights how operating system security plays a critical role in cybersecurity.
